1. !DOCTYPE html:
Indica que el código a trabajar se conforma con código HTML.

2. html lang="es":
Indica el idioma en el que se trabajara el código

3.head:
Encabezado de la pagina web y dentro se albergan varias etiquetas complementarias

4. meta charset="UTF-8"
Librería para símbolos tildes etc.

5. meta name="viewport" content="width=device-width, initial-scale=1.0"
Esta etiqueta ayuda a que el contenido se visualice de manera adecuada en dispositivos móviles o pantallas mas pequeñas a una computadora



6. title DW 18012021 /title
Titulo o lo que aparecerá en la pestaña del navegador

7. link rel="stylesheet" href="style.css"
Es el estilo que se le dará a la pagina web en este caso CSS

8. link rel="shortcut icon" href="favicon.png"
Sirve para poder agregar iconos

9. /head
Cierre de la etiqueta de encabezado

10. body
Comienza el cuerpo del código, en esta área se agregara todo el diseño de la pagina web

11. h1 DW 2021 - Actividad 2 /h1
Muestra un titulo

12. script src="script.js" /script
Tecnología de Javascript

13. /body
Cierre de el cuerpo de la pagina web

14. /html
Cierre del código html
